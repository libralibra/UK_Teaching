Installing Python

Installing Python for Windows

  • Python website: https://www.python.org>
  • Download the Windows x64 installer
  • Make sure to enable 'Add Python to PATH' when installing
  • Note: The latest version on the website is recommended unless you have a specific reason, such as certain 3rd-party libraries that only work with a particular version of Python.

Getting Python for Other OSes

  • Mac OSX:
    • Comes with an outdated version of Python, not recommend
    • Download the latest version from https://www.python.org>, or install using homebrew
  • Linux:
    • May come pre-installed depending on your selection of distros
    • If not, check the package manager of your distribution
    • Note: some distros do separate py2 and py3

Choosing and IDE

  • IDLE: Integrated Development and Learning Environment
    • Comes with Python by default
    • "Basic" functionalities: syntax highlighting, limited autocomplete
  • Visual Studio Code: free and open-source, cross-platform, general purpose IDE
    • Download: https://code.visualstudio.com/Download
    • Python extension among others
    • Useful features: enhanced syntax highlighting, autocomplete, auto importing, linting, basic debugging, etc.

Choosing and IDE (cont.)

  • PyCharm: free and open-source, cross-platform IDE made for Python
    • Download: https://www.jetbrains.com/pycharm/
    • Sign up for an educational account at https: //www.jetbrains.com/community/education/ to get the Professional version for free
    • Fully-featured: advanced debugging, package management
  • Spyder IDE: free and open source, Scientific Python Development Environment
    • Download: https://www.spyder-ide.org/
    • Written in Python, for Python developers
    • Matlab-like GUI, pre-installed with Anaconda

How to run Python code?

  1. Type code into the interactive interpreter directly (e.g. IDLE, IPython)
  2. Write code to a .py file in an IDE and execute
  3. use a more advanced interactive environment like Jupyter Notebook

Python Basics

Hello World

                            print('Hello world!')
  • Code does not have to be inside a class or function
  • No semicolons
  • String laterals starts and ends with ', ",''', or """
  • print is a built-in function

Comments

# This is a comment line
# No block comment command
# Multi-line comment works in the same way as a single line comment

Variables

a = 1
b = 2
c = a + b

Variables doesn't need to be declared

Variables

x = 7
x = 12.30
x = "Hello"
x = True

Variables can hold values of any type

if Statement

if 0 < x < 40:
    print('Fail')
elif x < 60:
    print('Pass')
elif x < 70:
    print('Merit')
elif x <= 100:
    print('Distinction')
else:
    print('Wrong number')
  • Indentation matters - the biggest feature of Python!
  • Note the colons and the lack of parentheses
  • Note the elif - the shorthand of else
  • Note the syntax of 0 < x < 40

List

data = ['apple', 2, 3.5, False]
print(data[0])
data[-1] = 6
  • Can store values of any type within []
  • Compare to c-style languages: int a[2]; int b[2][3];
  • One of the most, if not the most, important data types in Python.
  • Can use negative index (counting from right to left starting with -1)

Dictionary

data = {'name': 'Alice',
        'age': 19, 
        'courses': ['COMP201','COMP270'], 
        'score': 66.8}
print(data['name'])
data['age'] = 20
  • Another important data type in Python defined by {}
  • Items in dict are ordered, changeable, and no duplicates
  • Very useful for storing key:value pairs

for Loop

for x in data:
    print(x)

Works like foreach in C# or JavaScript

for i in range(5):
    print(i)
  • range(n) returns the numbers in the range of $[0,n)\> or\> [0,n-1]$
  • The other way of a for loop using index
  • Python doesn't have the C-style for loop (for(int i=0;i<10;i++)...)
  • continue and break works in the same way as other programming languages

while Loop

i = 0
while i < 20:
    i += 1
    if i % 2 == 0:
        continue
    elif i == 9:
        break
    print(i)
  • Similar to for loop but checks for a condition
  • Remember to modify the counter within the loop of it runs forever
  • continue and break works in the same way as other programming languages

Function

def add(a, b):
    return a + b 
def foo(txt):
    print('The input information is:', txt)
def bar(var1, var2='default parameter value'):
    print(var2)
    return None
  • Use keyword def to define a function
  • Can return any value (int, float, string, list, dict, ...) or nothing
  • Default values can be defined in function definition

Function as Value

def add(a, b):
    return a + b 

print(add(5, 8))

f = add 
print(f(10, 20))

x = lambda a, b: a + b
print(x(2, 5))
  • Functions are values - can be assigned to variables
  • Use lambda to define anonymous functions

Class

class Adder:
    def __init__(self, a, b):
        self.a = a 
        self.b = b 
    
    def add(self): 
        return self.a + self.b
  • __init__ is the constructor - must-have for any class
  • self is equivalent to this
  • self has to be referred explicitly - different from this
    - self must be the first parameter of any method/function

  • A useful way of commenting your code
  • The built-in function help searches for it

List Comprehension

data = [2, 4, 5, 6, 15, 20, 36]
square_data = [x*x for x in data]
less_than_20 = [x for x in data if x < 20]
double_if_greater_than_20 = [x*2 for x in data if x >= 20]
  • Python's be-loved syntactic sugar (dict has a very similar usage)
  • Provide a shorthand for loop
  • Follow the form of the mathematical set-build notation
  • Similar to LINQ queries in C#

File I/O: Reading Data from a File

fin = open('data.txt','r')
for line in fin.readlines():
    print(line)
fin.close()
  • Open a file in r mode for reading data
  • The built-in .readlines() can return a string list of the lines
  • The file need to be closed afterwards
  • Can use rb to read binary files

File I/O: Writing Data to a File

fout = open('data.txt','w')
for i in range(10):
    fout.write(f'This is line {i}')
fout.close()
  • Open a file in w mode for writing data
  • The built-in .write() can write a string to the file, .wirtelines() writes a list of strings in one go
  • The file need to be closed afterwards
  • Can use wb mode to read binary files, or a mode to append to a file

File I/O: Best Practice

with open('data.txt','w') as fout:
    fout.writelines(str_list)
# other code blocks ...
  • Use with ... as ...: clause to safely open a file (in both reading and writing modes
  • No need to call .close() function any more
  • The file will be closed automatically as long as the execution exits the with scope

Exception

try:
    result = 10 / 0
except ZeroDivisionError:
    result = "Error: Division by zero"
else:
    result = "Result: " + str(result)
finally:
    print(result)
  • Use try...except...(else)...(finally)... to handle exceptions
  • try must be accompanied by except
  • else and finally are optional

Python Code Structure

#! /usr/bin/env python3
# Description of script

def main():
    pass

# this will ensure that code will not run unless run as a script (eg, from the cli)
if __name__ == "__main__":
    main()
  • The 1st line is not compulsory, designed for Linux/Unix/Mac OS
  • You can leave comments after the leading #
  • Functions are defined by keyword def
  • It's a good practice to always include the __name__ check wrapper

Pygame

  • Cross-platform Python modules for game development
  • Includes features for managing multimedia elements using SDL (Simple DirectMedia Layer)
  • Provides functionality for capturing and responding to user input
  • Allows low-level control over the game loop

The Main Game Loop

Most main loops in games follow the same basic pattern:

repeat
    handle events
    update game state
    draw graphics
    maintain frame rate
    swap buffers
until "quit" signal received
